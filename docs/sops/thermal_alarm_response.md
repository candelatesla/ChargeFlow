# SOP: Thermal Alarm Response

Use this procedure when charger temperature alarms trigger, cooling warnings persist, or power output is throttled unexpectedly.

1. Review recent load, ambient weather, and connector temperature telemetry.
2. Inspect fan operation, vent blockage, and cable cooling behavior.
3. If the connector temperature remains elevated, remove the charger from service.
4. Apply the approved thermal reset process and monitor two stable charging cycles.
5. Replace cooling or cable components when repeat alarms occur within seven days.
6. Record all thermal findings, replaced parts, and validation steps in the ticket notes.
