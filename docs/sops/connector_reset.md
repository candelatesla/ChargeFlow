# SOP: Connector Reset Workflow

Use this procedure when a charger reports connector handshake issues, latch failures, or repeated customer complaints about incomplete sessions.

1. Confirm the affected station ID and port number from the most recent ticket or telemetry alert.
2. Place the charger port into maintenance mode before beginning physical inspection.
3. Inspect the connector head for debris, bent pins, cable strain, and latch damage.
4. Clean the connector and perform a controlled controller reboot.
5. Run two test sessions before returning the port to service.
6. Document the root cause, parts replaced, and customer impact in the maintenance ticket.
